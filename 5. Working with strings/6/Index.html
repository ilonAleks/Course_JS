Task 6

You need to implement a program that will request 2 values from the user through prompt

  1. Any text in prompt, send the message “Enter the text”. Save this value in the userText variable.
  2. The word that needs to be found in the text userText. In prompt, send the message “Enter the word from the text”. Save the value that the user writes into a variable wordFromText.
Also consider that there should not be unnecessary spaces at the beginning and at the end (use trim).

You need to find the word index wordFromText in line userText and save this index in variable indexOfWord. After cut the line userText Starting from the zero index and ending with the index, stored in the variable indexOfWord.

Using alert, display the message “The result: string”, where string - the final cut line.
