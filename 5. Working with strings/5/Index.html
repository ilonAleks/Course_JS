Task 5

You need to create a program that, according to the numbers introduced by the user, will cut this line.
through prompt, you need to request the following data:

  1. The line that needs to be cut. Save it in a userString variable. Also note that the line should not be extra gaps at the beginning and at the end (use trim). In prompt, send the message “Enter the text for trim”.
  2. The number that will indicate the index from which the trim will begin. In prompt, Send a message “Enter the index from which you need to start pruning the line. ” Save this value in the variable startSliceIndex.
  3. The number that will indicate the index from which the pruning of the line will end. In prompt, Send a message “Enter the index that need to finish the pruning of the line.” Save this value in the endSliceIndex variable.
After, bring the message alert “Result: string” through alert, where string is the final cut line.

Note: note that prompt always returns the type of data string.
