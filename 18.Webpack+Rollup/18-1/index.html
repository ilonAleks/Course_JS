Task 18-1

You have been assigned to a project where you are the main developer. No one has written the code yet, and therefore your task is to make the basic configuration of the project from an empty folder.

To do this, use rollup.js. Now in the project folder create 3 files: index.html, index.css and index.js. Add some basic html markup to index.html. Next, create a package.json file with the npm init -y command. Also in index.js add a line of code console.log('Hello World!');

Now, following the rollup.js documentation (https://rollupjs.org/guide/en/) tutorial video, install rollup as a package and create the rollup.config.js config file. In rollup.config.js , add the basic configuration to run the project. Entry, output and watch properties will suffice for this. Don't add any plugins yet.

Then in the package.json file create 2 commands "build" and "start".
"build": "rollup -c",
"start": "rollup -c -w"
Then run the "start" command. You should generate a JS file, the name of which you specified in the “output” property in the rollup.config.js file. If the file is not generated, then you may have done something wrong and you have an error. In this case, you need to find it and fix it.
