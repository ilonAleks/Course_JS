Task 18-3

You have successfully initialized the project on rollup.js. The whole development team is happy with you, because everyone can now start implementing the functionality.

Your boss has figured out that you're a pretty good developer and is asking you to help start another project. The only thing is that the requirement from the customer is the following: you must use only webpack.js.

Your job now is to start from an empty folder and initialize the project with webpack.js.

In the project folder, create 3 files: index.html, index.css and index.js. Add some basic html markup to index.html. Next, create a package.json file with the npm init -y command. Also in index.js add a line of code console.log('Hello World!');

Now, following the webpack.js documentation (https://webpack.js.org/guides/installation/), video tutorial, install webpack as a package and create the webpack.config.js config file.

In webpack.config.js , add the basic configuration to start the project. Entry, output and mode properties will suffice for this. Don't add any plugins yet.

Then add the “start” command to the package.json file.
"start": "webpack --config webpack.config.js"
Then run the "start" command. You should generate a JS file, the name of which you specified in the “output” property in the webpack.config.js file. If the file is not generated, then you may have done something wrong and you have an error. In this case, you need to find it and fix it.
