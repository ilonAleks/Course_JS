Task 18-2

You are well done! You have initialized the project. But that's not all, as you also need to take into account the following things:

  1. Connecting styles;
  2. Working with pictures;
  3. Code support by other browsers;
  4. Run the project on the local server (localhost).
So that's what you're going to do now. Several plugins need to be installed as devDependencies.
  1. Use “rollup-plugin-styles” to include styles: https://www.npmjs.com/package/rollup-plugin-styles
  2. To work with "rollup-plugin-img" images: https://www.npmjs.com/package/rollup-plugin-img
  3. To translate JavaScript into code that even older browsers support, use “@rollup/plugin-babel”: https://www.npmjs.com/package/@rollup/plugin-babel
  4. To run the project on a local server, you need the “rollup-plugin-serve” (https://www.npmjs.com/package/rollup-plugin-serve) and “rollup-plugin-livereload” (https://www. npmjs.com/package/rollup-plugin-livereload).
Then create an assets folder and add any picture with the image of the JavaScript programming language to it. Now in the index.js file, you need to add two elements to the DOM:
  1. h1. This element should contain the inscription “I love JavaScript”;
  2. img. This element should display an image from the assets folder.
In addition, in index.css, add any styles to your taste. The main thing is that this file is not empty.

Then run the “start” command and use the script tag to include the js file generated by the rollup.js package into index.html.

As a result, you should have a picture displayed, all styles applied, and a local server should open. If something went wrong, then you may have an error. In this case, you need to find it and fix it.
