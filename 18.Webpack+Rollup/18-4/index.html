Task 18-4

You are well done! Even webpack.js is up to you! Now you need to deal with the same points that were when using rollup.js:

  1. Connecting styles;
  2. Working with pictures;
  3. Code support by other browsers;
  4. Run the project on the local server (localhost).
You need to install some plugins as devDependencies and add some code to the config file.
  1. Use “css-loader” (https://www.npmjs.com/package/css-loader) and “style-loader” (https://www.npmjs.com/package/style-loader) to include styles.
  2. To work with images, add this code to webpack.config.js
    module: { rules: [ { test: /\.(?:ico|gif|png|jpg|jpeg)$/i, type: 'asset/resource', }, { test: /\.(woff(2)?|eot|ttf|otf|svg|)$/, type: 'asset/inline', }, ], },
  3. To translate JavaScript into code that even older browsers support, use the "babel-loader" (https://www.npmjs.com/package/babel-loader).
  4. To run the project on a local server, you will need the “webpack-dev-server” plugin (https://www.npmjs.com/package/webpack-dev-server). To configure it, add this code to webpack.config.js: devServer: { open: true compression: true port: 4000 },
Also, in order for webpack to process HTML and automatically include JavaScript in it, you need to install the html-webpack-plugin (https://www.npmjs.com/package/html-webpack-plugin). Add this code to webpack.config.js to configure this package: plugins: [ new HtmlWebpackPlugin({ template: 'index.html' }), ] Then add the code that you wrote when using rollup.js to the index.js file. Also don't forget to add the image to assets and copy the index.css file with styles.

In addition, replace the commands in package.json with the following:
"start": "webpack serve",
"prod": "webpack"
Then run the "start" command. If you did everything correctly and followed the instructions in the plugin installation links, then you should have a local server open. You should also have an image from assets displayed and all styles applied. If something went wrong, then you may have an error. In this case, you need to find it and fix it.
