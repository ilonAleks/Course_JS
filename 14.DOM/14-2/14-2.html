Task 14-2

You, together with the development team, are creating an application to control tasks. Now the team has only made a layout to make the application beautiful. You were instructed to draw all the data on the tasks using JavaScript code (select the method yourself). You have an array of TASKS and an HTML-glue, according to which you need to create elements for each task.

Add all HTML elements with tasks in the tag with class “tasks-list”.

HTML-Template for the task:

<div class="task-item" data-task-id="1"> <div class="task-item__main-container"> <div class="task-item__main-content"> <form class="checkbox-form"> <input class="checkbox-form__checkbox" type="checkbox" id="task-1"> <label for="task-1"></label> </form> <span class="task-item__text">View new JavaScript lesson</span> </div> <button class="task-item__delete-button default-button delete-button" data-delete-task-id="5">Delete</button> </div> </div> Task array:

const tasks = [ {id: 1, completed: false, text: 'View new JavaScript lesson'}, {id: 2, completed: false, text: 'Perform a test after a lesson'}, {id: 3, completed: false, text: 'Perform HW after a lesson'}, ]
ID data should be used in data-task-id, in id at input and in for at label. The value of the property text should be added to the tag span with the class “task-item__text”.

Note: The attribute for in JavaScript code is written as htmlFor.
