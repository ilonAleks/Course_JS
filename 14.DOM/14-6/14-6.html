TAsk 14-6

You are congratulated by the customer and the development team on successful work on the project.You are well done!They showed themselves well.

But then one of the developers fell ill and there is no one else to fulfill his task. Therefore, they asked you to replace a colleague.

You need to to realize the change of topic with dark to bright and vice versa. The topic should switch when pressing the “Tab” button. Initially, the topic is bright.

When changing the topic, you need to change styles for the following elements:

  1. An element with a tag body. If the topic is dark, then the Background property should be the value “#24292E”, otherwise - “initial”.
  2. All elements with a class “task-item”. If the topic is dark, then the Color property should be “#ffffff”, otherwise - “initial”.
  3. To all elements with a button tag, you need to add a value “1px solid #ffffff”, If the topic is dark, otherwise - “none”.
To solve this problem use properties style HTML elements.
