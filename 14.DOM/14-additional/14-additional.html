A falling list is a popular element on a modern website. Have you ever wondered how such an element is created? Now your task will be the implementation of the drop -down list on the clean JavaScript.

You already have styles and an HTML challenger for a drop-down list. Therefore, you just need to add logic. You need to create a template for a drop -down list through JavaScript using document.createElement.

The trim of the drop -down list:

<div class="select-dropdown select-dropdown--123"> <button class="select-dropdown__button select-dropdown__button--123"> <span class="select-dropdown select-dropdown--123"> Select element</span> </button> <ul class="select-dropdown__list select-dropdown__list--123"> <li class="select-dropdown__list-item" data-value="1">JavaScript</li> <li class="select-dropdown__list-item" data-value="2">NodeJS</li> <li class="select-dropdown__list-item" data-value="3">ReactJS</li> <li class="select-dropdown__list-item" data-value="4">HTML</li> <li class="select-dropdown__list-item" data-value="5">CSS</li> </ul> </div> To implement the drop -down list create a class CustomSelect. The designer of this class accepts 2 parameters:
  1. id. Unique list identifier.In the HTML, ID template was substituted as the text “123”. Therefore, the ID parameter must be substituted in place of this text.
  2. options. An array of choice for a drop -down list. The array consists of objects with Value keys (Option) and Text (Option text that the user sees).
In class CustomSelect All methods and fields except render must be private. Public method render Accepts the parameter container. In this container The whole layout from the template list template should be added.

To implement the opening and closing of the list, you need to hang the events handler “click” on the element with the class “select-dropdown__button”. The list is open when the element with a tag ul has class “active”.

To implement the choice of a certain option From the drop -down list, it is necessary to hang an event handler “click” on the all elements with tag li or use the delegation of events (Choose that you are more convenient). When clicking, get a value stored in the attribute “data-value”. Substitute this value as a text in an element with a class “select-dropdown”. In addition to the element li, on which the click occurred, add the class “selected”, So that it becomes active and illuminated by another color. Class “selected” can only have one element with a tag li.

To receive the current selected value create a private field currentSelectedOption and getter for him, which will be called selectedValue. Getter selectedValue must return the object of the selected Option from the drop -down list (an object from an array options).
