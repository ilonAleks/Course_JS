Task 10-2

You need to create a function giveJobToStudent, which accepts 2 parameters:

  1. student. Object containing information about the student.
  2. jobName. The name of the new work of the student.
Function giveJobToStudent displays the message to the modal window “Congratulations! Student fullName has a new job! Now he jobName”, где fullName - This is the name of the student, а jobName - the name of the new work of the student.

Function giveJobToStudent must return a new object of a student in which all the keys to the object will be student And the new job key with the parameter value will also appearjobName.

For the test of the function giveJobToStudent use the following code:

const student = {
fullName: 'Maxim',
experienceInMonths: 12,
stack: ['HTML', 'CSS', 'JavaScript', 'React'],
}

const updatedStudent = giveJobToStudent(student, 'Web developer');
