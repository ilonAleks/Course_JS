Task 10-5

Have you ever played the lottery? The lottery is a game in which the winner is randomly determined and give him win. Now your task will be to develop logic for such a game.

You need to create a function getWinner, which accepts 2 parameters:

  1. applicants. An object in which the keys are the numbers of people by which random selection will be made, and meanings - these are objects of candidates for winning in the lottery.
  2. winnerObject. This is an object that stores only 1 Prize key that stores the winnings in the lottery.
You need to randomly choose a winning number (random key in the object applicants) and return from functions getWinner an object in which properties will be stored from winnerObject and the object of the winner.

To obtain random value in the range, use the following function:

function getRandomNumberInRange(min, max) {
  return Math.floor(Math.random() * (max - min)) + min;
}

Example of the return result of the function getWinner:

const todaysWinner = {
  prize: '10 000$',
}

const winnerApplicants = {
  '001': {
    name: 'Maxim',
    age: 25,
  },
  '201': {
    name: 'Svetlana',
    age: 20,
  },
  '304': {
    name: 'Catherine',
    age: 35,
  },
}

const resultWinner = getWinner(winnerApplicants, todaysWinner);
console.log('resultWinner', resultWinner); // { prize: '10 000$', name: 'Catherine', age: 35 }
