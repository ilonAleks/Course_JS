Task 10-6

Imagine that you are developing a program for issuing coupons for a local hospital. Before that, the coupons were written out manually and you need to optimize this task.

You need to create a function giveTalonsInOrder, which accepts 2 parameters:

  1. patients. An array of objects, each of which stores information about the name of the patient and his unique number id.
  2. orders. An array of unique id numbers, which indicates the order in which patients should stand in line.

The function should return a new array in which objects from an array patients will be sorted by id from array orders.

Look at the possible result of the function giveTalonsInOrder

const ordersArr = [4, 2, 1, 3];
const people = [
  { id: 1, name: "Maxim" },
  { id: 2, name: "Nicholas" },
  { id: 3, name: "Angelina" },
  { id: 4, name: "Vitaly" },
];

const result = giveTalonsInOrder(people, ordersArr);
console.log('result', result);
