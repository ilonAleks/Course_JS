Task 10-4

Imagine that an investigation is being conducted in the police station. Several crimes were committed. The police have other more priority tasks, so they asked you to write a program that will be Calculate the criminal according to already known data.

Your task is to create a function getKiller. getKiller accepts 2 parameter:

  1. suspectInfo. This is an object in which the keys are people suspected of a crime, and the values are masses of people, who were seen by the suspects on the day of the murder.
  2. deadPeople. This is an array with the names of people who were killed by the criminal.
The criminal is the one who saw all the dead people on the day of the murder.Function getKiller must return the name criminal.

Examples of function results getKiller:

getKiller(
  {
    'James': ['Jacob', 'Bill', 'Lucas'],
    'Johnny': ['David', 'Kyle', 'Lucas'],
    'Peter': ['Lucy', 'Kyle'],
  }, ['Lucas', 'Bill']
); // Killer is James

getKiller(
  {
    'Brad': [],
    'Megan': ['Ben', 'Kevin'],
    'Finn': [],
  },
  ['Ben']
); // Killer is Megan
