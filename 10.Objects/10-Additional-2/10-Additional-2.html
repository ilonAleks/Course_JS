Task 10-Additional-2

A customer came to you, who is the owner of one of the game web platforms. He wants you to develop for His site is a new game that will conquer the hearts of many users.

The game has 2 players : hero and enemy. They will fight each other. Each player has a health scale, which initially, it is 100. With each hit, the opposite player is taken away by 10 units of health. The one wins. Who is the health of more than 0.

Now you need to create a function startGame, which will accept 2 parameters:

  1. heroPlayer. The object of the player that contains the properties of NAME - the name of the hero; Health - a health scale that initially equal to 100; Heatenemy - a function that takes the Enemy object and takes away from the Enemy 10 object Health units (Health key).
  2. enemyPlayer. The object of the enemy, which contains the properties of NAME - the name of the hero; Health - a health scale that initially equal to 100; Heathero - a function that takes the Hero object and takes away from the Hero object 10 units Health (Health key).

Inside the function startGame you need to randomly generate a number from 0 to 1. If 0 falls, then need call the method heatEnemy at the object heroPlayer, if 1 - then heatHero at enemyPlayer.

For this so that the function generates random values several times and players fight until someone has will end life, it is recommended to use the While cycle, which will be executed until any player has a Health property Less or equal to zero.

After completing the cycle, it is necessary determine which player won and display a message through alert “${name} won! He has left ${health} health”, where name and health - the values of the properties of the winner player.

To generate random values use the following function.

function getRandomNumberInRange(min, max) {
  return Math.floor(Math.random() * (max - min)) + min;
}

To verify performance functions startGame Use this code. There are no methods in it heatEnemy and heatHero, to you it is necessary to implement them themselves.

}
