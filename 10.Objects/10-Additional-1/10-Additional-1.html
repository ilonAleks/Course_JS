Imagine that you are developing an online food delivery store. You were set the task of calculating the total amount all goods in the basket.

You need to create a function getTotalPriceOfShoppingBag, which will take 1 parameter shoppingBag. shoppingBag - This is an array of products in the basket consisting of objects in each of which information about the name of the product (product) and the quantity of the product in the basket are stored (quantity).

Also the online store has a global object in which all the necessary information is stored about each unit product:

const groceries = {
  "Orange Juice": {
      price : 1.5,
      discount: 10,
},
  "Chocolate": {
      price : 2,
      discount : 0,
},
// more items...
}

Function getTotalPriceOfShoppingBag should return the total cost of all goods in the basket, taking into account discounts and given the number of products indicated by the client.The final value should be rounded to hundredths.It's possible do it with toFixed (https://developer.mozilla.org/...)

Look at the possible return value of the function getTotalPriceOfShoppingBag:

const shoppingBag = [
  { product: 'Chocolate', quantity: 3 },
  { product: 'Orange Juice', quantity: 23 },
]

const totalPrice = getTotalPriceOfShoppingBag(shoppingBag);
console.log('totalPrice', totalPrice); // Returns 37.05
