Task 10-1

A massif of users is given to you users. Each of them has a property status, which may be equal or “online”, or “offline”.

const users = [
  {
    username: 'David',
    status: 'online',
    lastActivity: 10
  }, {
    username: 'Lucy',
    status: 'offline',
    lastActivity: 22
  }, {
    username: 'Bob',
    status: 'online',
    lastActivity: 104
  }
]

You need to create a new array onlineUsers, which will contain objects of only those users, which status is equal “online”.

After getting it out through alert message “Now online The following users: usersOnlineNames”, where usersOnlineNames -string in which the user names are displayed through a comma.

For code above, the result should be as follows: “The following users are online now: David, Bob”.
