Task 10-3

You need to create a function that will work with objects. Name it handleObject.

handleObject accepts 3 parameters:

  1. obj. The object with which the function will work.
  2. key. Object key.
  3. action. The action that we will perform over the object.

The Action parameter can be 4 values:
  1. ‘get’. If the Action is equal ‘get’, function handleObject should return the value of the key key in the object obj.
  2. ‘add’. If the Action is equal ‘add’, function handleObject should add a new key to the Object object and Assign the value of an empty line "".Also, from the function, it is necessary to return the updated objectobj.
  3. ‘delete’. If the Action is equal ‘delete’, function handleObject should remove the property of Key from the Obj object and Return the updated object.
If the Action is equal To any other meaning, function handleObject must return the object obj.
