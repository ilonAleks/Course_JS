Task 20-6

Напишите функцию, которая принимает в качестве аргумента массив и возвращает новый массив. Она должна сортировать все нечетные числа по возрастанию, но четные остаются на своих местах
Write a function that takes an array as an argument and returns a new array. It should sort all the odd numbers in ascending order, but the even numbers stay where they are.
oddSort([7, 3, 4, 9, 5, 2, 17]) // => [3, 5, 4, 7, 9, 2, 17]
