Task 9-1

You need to create a function getSumOfSequence, which takes one parameter number. This function should create an array from numbers from 0 to number. That is, if the number was transferred to 5, then the array should look as follows: [1, 2, 3, 4, 5]. Use the cycle to fill the array with elements for.

Function getSumOfSequence must return the sum of the first and last elements of the final array.
