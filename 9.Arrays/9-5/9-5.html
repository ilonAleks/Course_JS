Task 9-5

You have developed the main functionality that Alexey wanted to see on his website. He tells you great thank you because thanks to the site in the coffee shop, the number of sales has increased. Now Alexey wanted to raise the price for coffee, so that there are even more profit.

You initially have 2 arrays: coffees (keeps the names of coffee) and prices (keeps coffee prices). Indices for names coffee and prices are interconnected. Those, “Latte” costs 1.5 euros, “Cappuccino” - 1 euro, etc.

coffees = ['Latte', 'Cappuccino', 'Americano'];
const prices = [1.5, 1, 2];

You need to create a new array updatedPrices, which will contain the same numbers from the array prices, but only increased by 0.5.

After raising prices, you need to notify the coffee houses about this, so bring for each type of coffee message through Alert “Coffee Name now stands Price Euro”, where NAME is the name of coffee, and Price is an updated price for Coffee.

Conditions:

  1. It is necessary to use the methods of MAP massifs and forEach
