Task 12-2

You have 2 objects dog and bird, which contains their description.

const dog = {
  name: 'Charlie',
  type: 'Dog',
  makeSound() {
    return 'Woof-woof'
  }
}

const bird = {
  name: 'Peatya',
  type: 'Sparrow',
  makeSound() {
    return 'Chic-chiric';
  }
}

You need to create a function makeDomestic, which will display all the information about the animal using the message to the console “type named name says sound”, where name - The name of the animal, type - its type and sound - the result of the call of the function makeSound.

Also a function makeDomestic must return a new object of the animal, in which there will be all the parameters that were before that (name, type, makeSound), as well as a new parametr isDomestic, which is a type of data boolean and it shows whether the animal is domestic or not.

To indicate the value for the property isDomestic (true or false) you need to transfer it as a parameter to a function makeDomestic.

In addition, inside the function makeDomestic must only use this. That is you are allowed to turn to the properties of the animal only this.name or this.makeSound, but not bird.name or bird.makeSound.

You need to solve this problem in 3 ways: through bind, call and apply. Call the function makeDomestic 3 times using these methods. Using bind, function makeDomestic should work with the object dog, at apply and call - with an object bird. Do not forget to transfer the parameter isDomestic in a function when calling it.
