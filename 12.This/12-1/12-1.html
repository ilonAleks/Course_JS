Task 12-1

Imagine that you as a web developer were asked to develop a web platform that will track progress students in learning web development.

First you need to create an object student, in which there will be 3 properties:

  1. stack. The array of lines, where each line is a technology. Initially, the stack array should be equal [‘HTML’].
  2. level. Number data type. Level is responsible for the current level of the student and is originally equal 1.
  3. improveLevel. A function that first increases the value of the level property in a unit. If Level is 2, then you need to add to the end of the stack array ‘CSS’, if 3 - add to the end of the stack value ‘JavaScript’, 4 - ‘React’, 5 - ‘NodeJS’. If the Level value has become more than 5, then you need to display in the modal window through Alert Message “The student has learned all the technologies!”. Also a function improveLevel must return the updated object at the very end student.

In the end the value of the property stack After performing the code below it should be equal to ["HTML", "CSS", "JavaScript", "React", "NodeJS"].

student
  .improveLevel()
  .improveLevel()
  .improveLevel()
  .improveLevel()
  .improveLevel()

Note: a chain from a row of coming functions impoveLevel works like this because improveLevel returns an object student, and the student object has a method impoveLevel.

Look for an example:

const student = {
  improveLevel() {
    return this;
  }
};

student
  .improveLevel()
  .improveLevel()
  .improveLevel()
