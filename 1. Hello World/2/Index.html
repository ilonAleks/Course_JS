Задание 2

В предыдущем задании вы уже создали 2 файла и подключили index.js в index.html. Настало время поработать с JavaScript.
Для начала, откройте index.html файл в браузере. Рекомендуется использовать Google Chrome.
Сейчас, используя JavaScript, вам требуется создать простой таймер, который будет показывать обратный отсчет. Для этого выведете в консоль строчку “Запускаем таймер” с помощью функции console.log. Затем добавьте 6 функций alert, каждая из которых будет выводить число, начиная с 5-ти и заканчивая нулем. После вывода нуля, напишите строчку кода, которая будет выводить в консоль “Обратный отсчет закончен”.
