Task 5

Now you will work with the Prompt function. It displays a modal window with a text field in which you can Enter any text. PROMPT can be used to give an accurate answer to the question or ask the user to enter any data.
To get started, create a new folder called “Prompt”. Add 2 files to this folder: index.html and index.js. By Example of Task 1 Make the base marking html and connect the index.js file to index.html.

The following mathematical questions and answers are given to you:

  1. How much will be 2 + 2? Answer: If you answered 4, then you are right
  2. What is the root of 9? Answer: If you answered 3, then you are right
  3. How much will be 2 in the 5th degree? Answer: If you answered 32, then you are right
Now your task is to ask questions using Prompt, enter the answer in the text field and after Show answers (written above) using Alert.

PROMPT function syntax is as follows: PROMPT (“your text”);
