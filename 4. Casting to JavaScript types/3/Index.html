Task 3

When we try to transform not completely standard entities to other types of data, we sometimes get Unusual values.
For example, if we convert the ALERT function to the line, we will see the following:

console.log (string (alert));
// Returns Function Alert () {[NATIVE CODE]}

Now your task is in order to convert clearly to string, number and boolean the following entities, look to the result and analyze it:

  1. console.log
  2. { name: ‘Maxim’ }
  3. Symbol(‘key’)
  4. Number
  5. ‘’
  6. 0
  7. -10
  8. ‘-105’
