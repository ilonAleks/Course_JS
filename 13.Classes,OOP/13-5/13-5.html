Task 13-5

In the last task, you created a dictionary. The customer is satisfied with the work done and the development team too. Since you have already dealt with the technology of the dictionary, you were asked to implement another dictionary. The customer wants to create a new category of words “Difficult words”.

Use the code written in the last task. You need to create a daughter class HardWordsDictionary from the class Dictionary. Constructor in HardWordsDictionary take NAME parameter also accepts and calls the parent class constructor.

All methods that are in Dictionary, should remain unchanged in HardWordsDictionary, except one. You need to reduce the method add, which should work according to the same logic as add in Dictionary, only the object of the word should look as follows:

{
  word: 'word',
  description: 'description',
  isDifficult: true,
}

For class test HardWordsDictionary use this code:

const hardWordsDictionary = new HardWordsDictionary('Difficult words')
hardWordsDictionary.add('Amateur', 'anyone who is engaged in science or art without special training, possessing only superficial knowledge.')
hardWordsDictionary.add('Neologism', 'new word or expression, as well as the new meaning of the old word.')

hardWordsDictionary.add('Quantum', 'the indivisible part of any value in physics.')
hardWordsDictionary.showAllWords()

// Amateur - anyone who is engaged in science or art without special training, possessing only superficial knowledge.
// Quantum - the indivisible part of any value in physics.
