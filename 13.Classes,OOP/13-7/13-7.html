Task 13-7

Your company creates an internal product to facilitate control over the growth of its developers. You were instructed to realize such a product.

First you need to create a class Developer, the designer of which will take 3 parameters:

  1. fullName. The name of the developer.
  2. age. Developer age.
  3. position. The current position of the developer in the company (for example, Junior, Middle, Senior).

Initialize all parameters using this. Also create a field inside the designer technologies, which by default will be equal to an empty array.

Besides in class Developer you need to create 2 methods:
  1. code. A method whose body is originally empty (there is nothing in curly brackets).
  2. learnNewTechnology. This method takes 1 parameter Technology, which should be added to the end of the array technologies.

After the actions done you should get a template Developer, thanks to which you will create subsidiaries.

You now need to create 3 subsidiaries from class Developer
  1. JuniorDeveloper. The designer of this class accepts 2 parameters: fullName и age. Call the parent class designer and hand over these 2 parameters there. As the 3rd we have position. You need to indicate it by default. Write the meaning “Junior” as a 3rd parameter of a parent constructor call. Besides reinstall the method code, so that he leads the line into the console “Junior developer writes the code...”. Array technologies should contain the following technologies: 'HTML', 'CSS', 'JavaScript'.
  2. MiddleDeveloper. Do the same work in the constructor as JuniorDeveloper. Only in place position submit the meaning “Middle”. Method code in class MiddleDeveloper should take a line into the console “Middle developer writes the code...”. Array technologies should contain the following technologies: 'HTML', 'CSS', 'JavaScript', ‘React’.
  3. SeniorDeveloper. Do the same work in the constructor as JuniorDeveloper. Only in place position submit the meaning “Senior”. Method code in class SeniorDeveloper should take a line into the console “Senior developer writes the code...”. Array technologies should contain the following technologies: 'HTML', 'CSS', 'JavaScript', ‘React’, ‘NodeJS’.

Note: When you are decide the method code in subsidiaries, you use the principle of polymorphism.

Test the written classes using this code:

juniorDeveloper.code(); // Junior developer writes the code...
middleDeveloper.code(); // Middle developer writes the code...
seniorDeveloper.code(); // Senior developer writes the code...

console.log(juniorDeveloper.fullName, juniorDeveloper.age,
juniorDeveloper.position); // 'Anastasia', 20, 'Junior'
console.log(middleDeveloper.fullName, middleDeveloper.age,
middleDeveloper.position); // 'Igor', 25, 'Middle'
console.log(seniorDeveloper.fullName, seniorDeveloper.age,
seniorDeveloper.position); // 'Maxim', 30, 'Senior'
