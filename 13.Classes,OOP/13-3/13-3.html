Task 13-3

Your friend Artem is the owner of a car service. Now all applications for repairing machines are processed manually, which of course is not very convenient. Therefore, Artem asked you as a web developer to help in automation of this process.

You need to create a class CarService, in the constructor of which 2 parameters will be transmitted:

  1. name. The name of the car service.
  2. workingHours. The object in which the keys are stored from and till.
    from means the time of the start of the working day for the car service,
    till - End of the end of the working day.
    Знtime of time is recorded in format “h:mm” (for example, 9:00).
Initialize these parameters using this. Also, the parameter workingHours is optional. Therefore, create a value (data type object) default. This object should be called DefaultWorkingHours and be static. If the parameter workingHours not transferred, then substitute inthis.workingHours meaning from DefaultWorkingHours, which is equal:

static DefaultWorkingHours = {
  from: '9:00',
  till: '20:00',
}

Add to class a method called repairCar, which should take in 1 parameter carName (car name for repair, data typestring).

If carName not transmitted, then display the error You need to specify the name of your car in order to repair it” in the console through console.error. The function of the function should end with this.

If the parameter carName was transferred to repairCar, Then you need to compare the current clock with the time of the car service. If the current time (clock) is not included in the range of the car service, then display via alert message “Unfortunately, we are now closed. Come back tomorrow”. Otherwise withdraw the message “Now we will repair your car carName! Please expect” (carName - parameter of the method repairCar).

Note: When comparing the time, take into account only the clock, do not pay attention for minutes. Let the minutes always be a line “00”.

For the test of performance class CarService use this code:

const carService = new CarService('RepairCarNow', { from: '8:00', till: '20:00' });
carService.repairCar('BMW');

Hint:
In order to extract from the line “8:00”, you can use the watch value .split(':'), get the first element from the resulting array and transform it to the number.
