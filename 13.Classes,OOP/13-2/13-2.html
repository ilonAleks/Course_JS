Task 13-2

You need to create a class Person, constructor of which will take 2 parameters:

  1. name. Human name.
  2. age. The number of full years.
You also need to to implement the method compareAge in class Person.
compareAge accepts a class copy Person and compares the values of the full years.

This code must return the result in the following format. If one Person the number of years is more or equal than that of the other, then display the message through alert “name1 older than name2”. Otherwise “name1 younger than name2”.

Launch this code for the test of your class Person

const person1 = new Person('Maxim', 24);
const person2 = new Person('Svetlana', 36);
const person3 = new Person('Irina', 23);

person1.compareAge(person2); // Maxim younger than Svetlana
person2.compareAge(person3); // Svetlana older than Irina
person3.compareAge(person1); // Irina younger than Maxim
