Task 13-4

You were entrusted with the task on the project. It is necessary to develop a dictionary in which you can add words with their description, delete words and receive them.

To get started create a class Dictionary, the constructor of which takes 1 parameter name. Initialize this value with the help this. Also, the constructor should be declared a field words, which by default should be an empty object.

You need to class Dictionary add 3 methods:

  1. add. This method takes 2 parameters: word - word, type of data string; description - Description of the word, type of data string. Assign to the Words object by Word (the ADD method parameter) the value of the object in which data on the word and its description will be stored. An example of such an object:

    {
      word: 'word',
      description: 'description',
    }

    If the word already exists in the object words, You do not need to rewrite the word that has already been preserved earlier.
  2. remove. This method takes the key from the Words object, according to which you need to remove the word and deleys this word from the object.
  3. get. This method takes the key from the Words object. It is necessary to return the word found from the method according to the transmitted key parameter.
  4. showAllWords. Данный метод не принимает в себя никаких параметров. Its task is to bring all the words that are in the Words object to the console in format “word - description” (word - the word, description - description of the word).

To test the written class use this code:

const dictionary = new Dictionary('Explanatory dictionary');
dictionary.add('JavaScript', 'popular programming language');
dictionary.add('Web developer', 'a person who creates new services and sites or supports and complements the existing');

dictionary.remove('JavaScript');
dictionary.showAllWords();
// Web developer - a person who creates new services and sites or supports and complements the existing
