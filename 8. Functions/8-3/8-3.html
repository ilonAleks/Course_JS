Task 8-3

You need to create a function getDivisors, which takes the only parameter number . It is default should be equal to 1.

This function returns the number of dividers for the number of Number. If Number is less than zero or is not a whole (can be checked using a function Number.isInteger(number)), then take it to the modal window using Alert the message “Number must be a whole and more than zero!”.

Examples:

  1. getDivisors(4) = 3 // 1, 2, 4
  2. getDivisors(5) = 2 // 1, 5
  3. getDivisors(12) = 6 // 1, 2, 3, 4, 6, 12
  4. getDivisors(30) = 8 // 1, 2, 3, 5, 6, 10, 15, 30
