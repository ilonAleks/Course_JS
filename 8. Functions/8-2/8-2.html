Task 8-2

It is necessary to create a function getSumOfNumbers. Select any type of function when creating.

getSumOfNumbers accepts 2 parameters:number и type. Number parameter is the number by which will be counting the amount. That is, if the number 10 was transferred, then the function should calculate the amount from 0 to 10 (0 + 1 + 2 + ... + 10). Type parameter is responsible for the choice of numbers for calculating the sum.It can be 3 values: “odd”, “even” and “”. If Type is “odd”, then only odd numbers should include the amount, “even” - even numbers, an empty line “” ” - all numbers. By default, the Type parameter should be odd.

Function getSumOfNumbers should return the total amount using return.

Possible functions getSumOfNumbers:

  1. number = 10, type = ‘odd’. Returns 25.
  2. number = 10, type = ‘even’. Returns 30.
  3. number = 10, type = ‘’. Returns 55.
