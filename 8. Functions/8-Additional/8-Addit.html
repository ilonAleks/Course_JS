Task 8 Addition

Prior to this, you transmitted to the function such data as String, Number, Boolean. But in JavaScript function parameter You can convey anything, even a function. A function that is transmitted as a parameter to another function is called Callback. And you will work with this now.

First, create 2 functions showSuccessMessage, which takes the parameter message and leads it to console through console.log; showErrorMessage, which assumes the Message parameter and takes it to the console through console.error (console.error it displays a message to the console in red). These functions will be our future Callbacks.

Now you need to create another function called checkTextOnErrorSymbol. With it, we will look for text is prohibited and display an error into the console if such a symbol was found.

checkTextOnErrorSymbol accepts 4 parameters:

  1. text, Type of data string. It is responsible for the text in which we will look for the prohibited symbols
  2. errorSymbol, Type of data string. It is responsible for the prohibited symbol.
  3. successCallback. This function, which will display a successful message if there were no prohibited characters found.
  4. errorCallback. This function, which will display an error into the console, if a prohibited symbol has been found.
If the prohibited symbol was found , then call the flask errorCallback and pass the message into it “Found a prohibited symbol "${errorSymbol}" under the index ${i}.” (replace errorSymbol on the prohibited symbol, i - on the index lines in which the prohibited symbol was discovered.

If not a single prohibited symbol was found in the line , then call the flask successCallback and pass it into it message "There are no prohibited characters in this text".

Check the performance of your function on this code:
const text = 'Hello! How are you! For a long time you and I have not seen each other.'
checkTextOnErrorSymbol(text, 'а', showSuccessMessage, showErrorMessage);

Для данного кода, в консоль должно быть выведено следующее:
Found a prohibited symbol "a" under the index 9
Found a prohibited symbol "a" under the index 15
Found a prohibited symbol "a" under the index 19
