Task 8-4

Now your task is to create a function checkQuestionAnswer, which accepts 2 parameters: question и correctAnswer. Parameter question - This is a question that will be asked to the user and will be transferred as parameter in the function prompt. correctAnswer - it is the correct answer to the question.

You need to get the meaning that the user will introduce into the text field, and to monitor it with the parameter correctAnswer. If the user has given the right answer, then display in the modal window through the alert message “Answer True”, otherwise - “The answer is incorrect”.

Also if the user introduces this line “aPpLe”, and the parameter correctAnswer will be equal "Apple", then the answer it must be counted as correct. That is, the answer should not depend on the register of symbols and on the number of spaces at the beginning and at the end of the answer.

When create a function , call this code to check it:
checkQuestionAnswer('Watermelon is a fruit or berry? ',' Berry');
checkQuestionAnswer('How many adults do an average of teeth?', '32');
checkQuestionAnswer('What is the smallest bird in the world? ',' Hummingbirds');
