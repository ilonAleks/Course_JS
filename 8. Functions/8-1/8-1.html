Task 8-1

As we have already dealt with, there are 3 types of functions: function expression, function declaration and the arrow function.

First of all, we will train them to create. The task is simple, you need to create 3 functions of three different types. Name it should be the following getName1, getName2, getName3.

Functions must be satisfied with the following conditions:

  1. 1 parameter should be transferred to each of these functions name.
  2. From each function, the value “name is name”, where name - the value of the transferred to the function parameter name.
After you need to call each function and display the value returned from it to the console.
