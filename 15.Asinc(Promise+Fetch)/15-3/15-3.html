Task 15-3

Imagine that you have a new project, where there is the following functionality: a display of a photograph that booted the fastest.

Create a function getFastestLoadedPhoto, which takes 1 parameter ids, which is an array of id parameters at the photo object. To get information about the photo, you need to use the following URL https: //jsonplaceholder.typico ... (1 - this is ID photos).Using an ids array, receive photographs that loaded the fastest at a fetch request. To solve the problem, use Promise.race.

For convenience, you need to add an span element with the text “Download ...” before loading the photo, and hide this element after downloading data.

To create a HTML element photo, use this template:

<li class="photo-item"> <img class="photo-item__image" src="https://via.placeholder.com/600/92c952" /> <h3 class="photo-item__title"> accusamus beatae ad facilis cum similique qui sunt </h3> </li> In src, img should be substituted with the value of the url property in photography, and in the tag h3 the value of the title property. Add the final photo to an element with id equal “data-container”.

For the test of the function getFastestLoadedPhoto Use this code:
getFastestLoadedPhoto([60, 12, 55])

Note: Be sure to not forget to prescribe catch blocks. In it simply take an error to the console with the help console.error.
