Task 15-additional

You, together with the development team, create a social network where you can share posts and leave comments on the post. So far, the project is at an early stage and therefore you need to work only with the drawing of the necessary elements.

You need to create a renderPost function, which will take 1 parameter postId. With postId you need to get a post. For example, using this URL https: //jsonplaceholder.typico... you will receive data about a post with id equal to 1.

After you receive post data, you need to get all the comments for a particular post. To, for example, to extract comments for a post with an id of 1, you need to use the following url https: //jsonplaceholder.typico...

To display information about post and comments on the page, use this template:

<div id="post" class="post"> <h1 class="post__title">The post's name</h1> <p class="post__text">Tht post's text</p> <b class="post__comments-text">Comments</b> <div class="post__comments"> <div class="post-comment"> <span class="post-comment__author"> maxim@gmail.com </span> <span class="post-comment__text"> laudantium enim quasi est quidem magnam voluptate ipsam eos\ntempora quo necessitatibus\ndolor quam autem quasi\nreiciendis et nam sapiente accusantium </span> </div> </div> </div> Each class is responsible for specific information:
“post__title” - the value of the property of the Title in the post;
“post__text” - the value of the Body property at the post;
“post-comment__author” The value of the Email property of the comment;
“post-comment__text” value of the Body property in comment;
