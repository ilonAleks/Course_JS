6-Additional

When we are registering, in most cases we ask for a password. And there are always some for him restrictions on the length or symbols. Now you will write a program that will check the password for validity, introduced by the user.

First you need to request a password from the user using prompt. Display the message “Enter password".

The password must satisfy the following conditions:

  1. The length of the password should be at least 3 characters and no more than 20 characters (including 20).
  2. The password must contain at least 1 capital (large) letter.
  3. The password must contain at least 1 digit.
If all conditions are satisfied, then withdraw alert Message “The password is valid. Welcome to account! ”, otherwise - “The password doesn't satisfy the conditions! Reload the page and try it again.”

Look at several possible results:
  1. “1234f” - does not satisfy the conditions
  2. “123456” - does not satisfy the conditions
  3. “1234F” - satisfies the conditions
  4. “12” - does not satisfy the conditions
  5. “JavaScript”- does not satisfy the conditions
  6. “JavaScript123” - satisfies the conditions
