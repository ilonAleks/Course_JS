Task 2

This code requests the names of new students 3 times and, if the student’s name was introduced, he is welcomed message “Welcome, newStudent!” (newStudent - student's name).

You need to rewrite this cycle for the cycle with while and do while, so that the logic of code execution and result remained unchanged.

for (let i = 0; i < 3; i +=1) {
    let newStudent = prompt('Enter the name of the new student!');
    newStudent = newStudent.trim();
    if (newStudent) {
        alert(`Welcome, ${newStudent}!`)
    }
}
