Task 16-3

Задание 16-3 You need to create a function renderAlbums, which will display data on albums in DOM. To obtain data, use this url https://jsonplaceholder.typicode.com/albums.

HTML-template for the album element looks as follows:

<li>The name of the album</li> Final HTML elements with information about the name of the album place in the tag with the class equal “data-container”.

For convenience you must add element span with the text “Download ...” before loading albums, and hide this element after downloading data.

Use exclusively async await and try, catch, finally. If an error has occurred when loading albums, then inside the tag is equal “data-container” place the text “There was an error in obtaining album data ...”.
