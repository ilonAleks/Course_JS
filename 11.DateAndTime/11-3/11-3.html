Task 11-3

In JavaScript, the date of the date can accept TimeStamp, i.e. the number of milliseconds from January 1, 1970 00:00:00 by UTC. According to this amount of milliseconds, an object of date is generated.

Thanks to this, to add a certain number of milliseconds, you can use the following entry: new Date(Date.now() + 5000), where 5000 - this is milliseconds. As a result, we will receive a date, which is 5000 milliseconds more current.

Using this knowledge, you need to create the addDays function that accepts 2 parameters:

  1. date. Date object.
  2. days. The number of days, the type of data is Number, by silence this parameter should be equal to one.

You need to the parameter date add a certain number of days that is transmitted to the parameter days. Function addDays must return the updated date.

Hint: do not forget that timestamp, which is transmitted to Date should be measured in milliseconds.
