Task 11-Additional

Your customer is a visa center. They have a customer database, but there is no script that would automate the process Visa issuing. Therefore, the visa center asked you to help you as a web developer to take this task for yourself.

You need to create a function allowVisa, which accepts an array of customers of the visa center. Each client - this is an object with parametersfirstName (client's name), lastName (client's lastname), criminalRecord (type of data boolean, is there a criminal record), passportExpiration (type of data string, date of expiration of the passport).

Function allowVisa must return the filtered array people so that it satisfies the next conditions:

  1. The current date should be less than the property passportExpiration.
  2. The client should not have a criminal record, the value of the property criminalRecord should be equal false.

For the test of the function , use this code:

const peopleWithVisa = [
  {
    firstName: 'Stasia',
    lastName: 'Ward',
    criminalRecord: true,
    passportExpiration: '19.06.2024',
  },
  {
    firstName: 'Elliot',
    lastName: 'Baker',
    criminalRecord: false,
    passportExpiration: '04.06.2023',
  },
  {
    firstName: 'Leighann',
    lastName: 'Scott',
    criminalRecord: true,
    passportExpiration: '31.07.2023',
  },
  {
    firstName: 'Nick',
    lastName: 'Pop',
    criminalRecord: false,
    passportExpiration: '31.12.2022',
  },
];

const result = allowVisa(peopleWithVisa);
