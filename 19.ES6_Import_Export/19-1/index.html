Task 19-3

In the tasks of this lesson, you will create from scratch a service for donating money for charitable purposes.

You initially have an initialized empty project. You need to download it and open it in a code editor.
Link to download

Install all packages with the "npm i" command and then run the project itself with "npm start".

One of the important parts of a project before starting development is the folder structure. In the root of the project, create the src folder, and in it 2 more - core and modules. The core folder will be needed for the global parts of the project, and modules, for the main blocks of the project (for example, a list of donations, a form for sending donations, etc.).

In the modules folder, create an app.js file and in this file you need to implement the App class. This class should have a run method that puts the text "Hello World" inside the body tag.

Import the default (export default) App class in the index.js file, instantiate the App class, and call the run method.
