Task 19-3

You are well done! You are creating a cool donation service step by step and you are doing well.
Now you will need to draw a list of all donations that users of the service will make.
In the modules folder, create a donate-list.js file and implement the DonateList class in it. The constructor of this class must accept an array of donates. This array consists of objects that have the keys date (the date the donation was created) and amount (the amount of the donation). Also in the DonateList class, implement the render method, which will render the HTML template below.

<div class="donates-container"> <h2 class="donates-container__title">List of donations</h2> <div class="donates-container__donates"> <div class="donate-item">July 6th 2021, 10:28:49 am - <b>$4</b></div> <div class="donate-item">July 6th 2021, 10:28:49 am - <b>$20</b></div> <div class="donate-item">July 6th 2021, 10:28:49 am - <b>$3</b></div> <div class="donate-item">July 6th 2021, 10:28:49 am - <b>$1</b></div> </div> </div> This HTML element is responsible for the donate element: <div class="donate-item">July 6th 2021, 10:28:49 am - <b>$4</b></div>


Here “July 6th 2021, 10:28:49 am” is the value of the date property from the donates array object, and “4$” is the value of the amount property from the donates array object.
Export the DonateList (export) class. Then import it into the app.js file.
Through the run method of the App class and with the help of DonateList, you need to place the entire layout of the donations list inside the element with the body tag.
To test the rendering of the list of donations, use this donates array.
const mockDonates = [
