Task 19-2

Congratulations! You have created the initial structure of your project and practiced import and export.

Now you will need to implement the form for creating donations. In the modules folder, create a donate-form.js file. In this file, you need to create a DonateForm class with a render method that will return the final form HTML element. Export the DonateForm class.

Inside the DonateForm class, you need to implement the HTML structure below using document.createElement. To separate the logic, you can create your own fields and methods in the class. Make them private whenever possible so that you can write pure OOP code and apply the principle of encapsulation.

<form class="donate-form"> <h1 id="total-amount">$28</h1> <label class="donate-form__input-label"> Enter amount in $ <input class="donate-form_donate-input" name="amount" type="number" max="100" min="0" required=""> </label> <button class="donate-form__submit-button" type="submit"> Donate now </button> </form> After you implement the HTML form template rendering logic, you need to import the DonateForm class in app.js. Through the run method of the App class and using DonateForm, you need to place the entire layout of the form inside an element with the body tag.
